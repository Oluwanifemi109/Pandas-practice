# Pandas-practice
A collection of hands-on Pandas projects covering data cleaning, EDA, merging, grouping, and aggregation.

#

### 📁 Projects
| Folder | Description | 
| ------- | ------------ |
| Data Cleaning | data-cleaning/Handling missing values, duplicates, and data types |
| EDA | Exploratory Data Analysis, summarizing and visualizing data |
| Merge-join-concatenate | Combining DataFrames using merge, join, and concat
| Groupby-aggregation | Grouping data and computing aggregate statistics |

#

🛠 Requirements

* Python 3.x
* Pandas
* Jupyter Notebook (optional but recommended)

#

Install dependencies:
bashpip install pandas jupyter
